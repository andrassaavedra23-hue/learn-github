Branches: Líneas paralelas
Experimenta sin miedo a romper nada

Un branch es como una línea de tiempo alternativa. Puedes crear una para experimentar, y si todo sale bien, la unes con la principal.

🌳 Analogía del árbol:
El main es el tronco principal. Los branches son ramas que crecen del tronco.

🎮 Crea branches visualmente

🧠 ¿Qué comando crea Y cambia a un nuevo branch?
A
git branch nuevo
B
git checkout -b nuevo
C
git merge nuevo
