¿Qué es Git y GitHub?
Entiende la diferencia en 30 segundos

Imagina que estás escribiendo un libro. Cada capítulo que terminas, lo guardas como capitulo-1.doc, luego capitulo-1-final.doc... 😵

Git resuelve esto. Es una máquina del tiempo para tu código.

💡 Analogía simple:
Git = Tu álbum de fotos (guarda versiones)
GitHub = Instagram (comparte tus fotos con el mundo)

🎮 Explora los conceptos

⚡
Git
Sistema que guarda el historial de cambios localmente.
☁️
GitHub
La nube donde se guardan y comparten tus proyectos.
📁
Repositorio
Una carpeta de proyecto rastreada por Git.
Commit
Una "foto" del estado de tu proyecto en un momento dado.
