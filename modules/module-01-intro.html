¿Qué es Git y GitHub?
Entiende la diferencia en 30 segundos

Imagina que estás escribiendo un libro. Cada capítulo que terminas, lo guardas como capitulo-1.doc, luego capitulo-1-final.doc, luego capitulo-1-final-final.doc... 😵

Git resuelve este problema. Es como una máquina del tiempo para tu código: guarda "fotos" de tu proyecto en diferentes momentos y te permite volver atrás si algo sale mal.

💡 Analogía simple:
Git = Tu álbum de fotos (guarda versiones)
GitHub = Instagram (comparte tus fotos con el mundo)

☁️
GitHub

Repositorio

📸
Commit
Una "foto" del estado de tu proyecto en un momento específico

A
